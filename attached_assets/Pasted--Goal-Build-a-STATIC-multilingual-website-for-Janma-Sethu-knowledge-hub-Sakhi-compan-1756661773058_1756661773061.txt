Build a **STATIC**, multilingual website for **Janma Sethu** (knowledge hub + Sakhi companion) using **React 18 + Vite + TypeScript + TailwindCSS + React Router**. **Everything must be clickable and open a page. No 404 page.**

**CRITICAL VISUAL RULE:** Reuse **only the theme** from the provided screenshot (light pastel gradient background, gradient span in H1, rounded white cards with soft shadows, pill trust badge, gradient buttons, system serif/sans). **Do not copy any screenshot words, nav labels, or structure** except the **look & feel**.

---

# 0) Project Setup

* Tech: React 18, Vite, TypeScript, React Router v6+, TailwindCSS.
* 100% static: **no network calls**. Use local state and `localStorage` only.
* Accessibility: semantic landmarks, labelled controls, alt text, focus rings, keyboard nav, **skip to content** link.
* Persist selected language (EN/HI/TE) in `localStorage` key **`js_lang`** and apply on first paint.

# 3) Multilingual (EN/HI/TE) & Chat Auto‑Language

Create `LanguageProvider` with `{ lang, setLang, t }`. `t(key)` reads from dictionaries and falls back to English. Persist `lang` in `js_lang`.

**Auto‑language for chat (client‑only):**

```ts
function detectScript(s: string): 'en'|'hi'|'te' {
  if (/\p{Script=Devanagari}/u.test(s)) return 'hi';
  if (/\p{Script=Telugu}/u.test(s)) return 'te';
  return 'en';
}
```

On submit, reply using the canned paragraph `chat_reply_<code>` below.
